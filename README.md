# 🪙 บอล แบงค์เก่า - ระบบสารบัญและประเมินราคาของสะสม (BallBankDirectory)

ระบบเว็บแอปพลิเคชันแบบ Full-stack สำหรับรวบรวมสารบัญข้อมูลและระบบประเมินราคารับซื้อเหรียญเก่า ธนบัตรเก่า พระเครื่อง และการ์ดเกมของประเทศไทย โดย **บอล แบงค์เก่า**

ตัวแอปพลิเคชันขับเคลื่อนด้วยขุมพลัง **React 19**, **Vite**, **Tailwind CSS v4**, **Express (Node.js)** และเชื่อมต่อกับ **Firebase Firestore** เพื่อจัดเก็บข้อมูลแบบเรียลไทม์ พร้อมการบูรณาการ **Gemini AI** เพื่อเป็นตัวช่วยวิเคราะห์ประเมินราคาและตอบคำถามผู้ใช้งานอย่างอัจฉริยะ

---

## 🌟 คุณสมบัติเด่นของระบบ (Key Features)

*   **🗂️ สารบัญรายการของสะสม (Directory List)**: หน้าแสดงข้อมูลเหรียญ ธนบัตร พระเครื่อง และของสะสมต่าง ๆ ที่กำลังเป็นที่ต้องการในตลาด พร้อมข้อมูลราคารับซื้อ รายละเอียด และรูปภาพประกอบที่ชัดเจน
*   **💬 ระบบประเมินราคาอัจฉริยะ (Smart Appraisal Chat)**: แชทบอทอัจฉริยะที่ใช้เทคโนโลยี **Gemini AI SDK** เพื่อให้ผู้ใช้สามารถปรึกษา ส่งคำถาม หรือข้อมูลของสะสมเพื่อประเมินราคาเบื้องต้นได้ทันที
*   **📝 ส่งคำขอประเมินราคาอย่างเป็นทางการ (Official Appraisal Submissions)**: ผู้ใช้งานทั่วไปสามารถส่งรูปภาพ รายละเอียด และช่องทางติดต่อกลับเพื่อให้แอดมินประเมินราคาอย่างเป็นทางการได้
*   **👑 แผงควบคุมผู้ดูแลระบบ (Admin Dashboard)**: ระบบการจัดการสำหรับแอดมิน เพื่อตรวจสอบรายการคำขอประเมินราคาอัปเดตสถานะ (อนุมัติ/ปฏิเสธ/รอดำเนินการ) เพิ่ม-ลบ-แก้ไขข้อมูลของสะสมในระบบสารบัญได้อย่างสะดวกสบาย
*   **📢 ช่องทางการติดต่อออนไลน์ (Online Channels)**: แสดงลิงก์ติดต่อช่องทาง Social Media อย่างเป็นทางการของ บอล แบงค์เก่า (Line, Facebook, YouTube, TikTok)

---

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

*   **Frontend**: React 19 (TypeScript), Tailwind CSS v4, Motion (Animations), Lucide React (Icons)
*   **Backend**: Node.js, Express (สำหรับ API Proxy และจัดส่ง Static Files)
*   **Build Tool**: Vite 6, Esbuild (สำหรับการ Bundle Server)
*   **Database**: Firebase Firestore (จัดเก็บข้อมูลประเมินราคาและข้อมูลสารบัญแบบ Real-time)
*   **AI Engine**: @google/genai (Gemini SDK Server-side API เพื่อความปลอดภัยของรหัสลับ API Key)

## 🚀 เริ่มต้นใช้งานภายในเครื่องคอมพิวเตอร์ของคุณ (Local Setup)

หากคุณดาวน์โหลดโค้ดนี้หรือโคลนจาก GitHub ไปยังเครื่องโลคัลของคุณ สามารถเริ่มรันโปรเจกต์ได้ด้วยขั้นตอนดังนี้:

### 1. ความต้องการของระบบ (Prerequisites)
*   ติดตั้ง **Node.js (เวอร์ชัน 18 ขึ้นไป)** และ **npm** บนเครื่องของคุณ

### 2. ดาวน์โหลดแพ็กเกจที่จำเป็น
เปิด Terminal ในโฟลเดอร์ของโปรเจกต์ แล้วรันคำสั่ง:
```bash
npm install
```

### 3. ตั้งค่าสภาพแวดล้อม (Environment Variables)
สร้างไฟล์ชื่อ `.env` ขึ้นมาที่ระดับนอกสุดของโฟลเดอร์ (Root) หรือคัดลอกไฟล์ต้นแบบจาก `.env.example`:
```bash
cp .env.example .env
```
จากนั้นเปิดไฟล์ `.env` และใส่ค่า API Key ของคุณ:
```env
GEMINI_API_KEY="ใส่รหัสกุญแจ Gemini API Key ของคุณที่นี่"
APP_URL="http://localhost:3000"
```

### 4. รันโปรเจกต์ในโหมดพัฒนา (Development Mode)
ใช้คำสั่งด้านล่างเพื่อเริ่มรัน Dev Server ทั้งฝั่ง Client และ Server (รันผ่านพอร์ต `3000`):
```bash
npm run dev
```
เปิดบราวเซอร์และเข้าชมได้ที่: [http://localhost:3000](http://localhost:3000)

### 5. ทำการ Build สำหรับ Production
สร้างไฟล์ที่บีบอัดและอัปเกรดความเร็วในการทำงานด้วยคำสั่ง:
```bash
npm run build
```
เมื่อ Build เสร็จสิ้น สามารถสตาร์ทเซิร์ฟเวอร์แบบ Production ได้ด้วยคำสั่ง:
```bash
npm start
```

## 🗄️ การตั้งค่าความปลอดภัยของ Firebase Firestore

โปรเจกต์นี้ใช้ Firebase Firestore ในการจัดเก็บข้อมูล กรุณาตรวจสอบให้แน่ใจว่าไฟล์ `firestore.rules` มีความปลอดภัย ตัวอย่างกฎความปลอดภัยพื้นฐานในระบบ:
*   ผู้ใช้ทุกคนสามารถอ่านข้อมูลในหมวดหมู่สารบัญได้
*   ผู้ใช้ทุกคนสามารถเพิ่มข้อมูลคำร้องขอประเมินราคาได้
*   เฉพาะแอดมินผู้ดูแลระบบเท่านั้นที่สามารถทำการลบ แก้ไข หรือดูข้อมูลส่วนตัวในระดับลึกได้

## 📤 วิธีการนำโปรเจกต์นี้ขึ้น GitHub จาก Google AI Studio

คุณสามารถนำข้อมูลโค้ดและโปรเจกต์นี้ทั้งหมดไปอัปโหลดขึ้น GitHub ของคุณได้อย่างง่ายดาย ผ่าน 2 วิธีหลักดังนี้:

### วิธีที่ 1: ใช้ปุ่มส่งออก (Export) ในหน้าต่าง Google AI Studio (แนะนำ 🌟)
1.  มองหาแถบ **เมนูตั้งค่า (Settings/Export)** หรือเมนูเครื่องมือในหน้าต่าง Google AI Studio Build
2.  เลือกคำสั่ง **"Export to GitHub"** หรือ **"Download ZIP"**
3.  หากเลือก **Export to GitHub**: ระบบจะให้คุณเชื่อมต่อบัญชี GitHub ของคุณ จากนั้นจะทำการสร้าง Repository ใหม่และอัปโหลดไฟล์โค้ดทั้งหมด (รวมถึงไฟล์ `README.md` และ `.gitignore` นี้) ขึ้นไปยัง GitHub ให้คุณโดยอัตโนมัติ!
4.  หากเลือก **Download ZIP**: ระบบจะดาวน์โหลดไฟล์โครงการทั้งหมดเป็นไฟล์ `.zip` จากนั้นคุณสามารถนำไปแตกไฟล์บนเครื่องคอมพิวเตอร์ของคุณ แล้วทำการผลักข้อมูลผ่าน Git CLI ไปยัง Repository ของคุณเองได้

### วิธีที่ 2: ใช้คำสั่ง Git บนเครื่องคอมพิวเตอร์ของคุณ (Git CLI)
หากคุณดาวน์โหลดเป็น ZIP และเปิดบนเครื่องคอมพิวเตอร์ของคุณแล้ว ให้ทำตามขั้นตอนต่อไปนี้ใน Terminal:
1.  เริ่มต้นใช้งาน Git:
    ```bash
    git init
    ```
2.  เพิ่มไฟล์ทั้งหมดเข้าในคิวเตรียม Commit:
    ```bash
    git add .
    ```
3.  บันทึกข้อมูล Commit แรก:
    ```bash
    git commit -m "Initial commit: ระบบสารบัญและประเมินราคาของสะสม บอล แบงค์เก่า"
    ```
4.  เชื่อมโยงกับ GitHub Repository ของคุณ (เปลี่ยน URL ให้เป็นของคุณ):
    ```bash
    git branch -M main
    git remote add origin https://github.com/ชื่อผู้ใช้ของคุณ/ชื่อเรโปของคุณ.git
    ```
5.  ผลักข้อมูลขึ้นไปยัง GitHub:
    ```bash
    git push -u origin main
    ```
